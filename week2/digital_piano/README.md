# Digital Piano using ESP32

This project implements a **Digital Piano** using ESP32, push buttons, and a passive buzzer. Musical notes are generated by varying the buzzer frequency, allowing the user to play melodies interactively.

## Features

* Four buttons generate musical notes:

  * **Do**
  * **Re**
  * **Mi**
  * **Fa**
* Pressing **Button 1 and Button 2 simultaneously** plays **Sol (392 Hz)**.
* Fifth button toggles between **Major** and **Minor** scales.
* Serial Monitor displays the currently played note and active mode.
* Sound generation using PWM-based tone output.

## Components Used

* ESP32 Development Board
* 5 Push Buttons
* Passive Buzzer
* Breadboard
* Jumper Wires

## Connections

* BTN1 → GPIO16
* BTN2 → GPIO17
* BTN3 → GPIO18
* BTN4 → GPIO21
* BTN5 → GPIO22
* Buzzer → GPIO25

## Output

The Serial Monitor displays messages such as:

```text
MAJOR MODE
DO
RE
MI
FA
SOL

MINOR MODE
DO
RE
MI
FA
```

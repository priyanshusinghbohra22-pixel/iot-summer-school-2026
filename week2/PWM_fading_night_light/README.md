# PWM Fading Night Light

This project demonstrates PWM-based LED brightness control using ESP32. A push button is used to switch between three lighting modes.

## Features

* **Mode 1:** Slow Breathing Effect
* **Mode 2:** Fast Pulse Effect
* **Mode 3:** SOS Pattern
* Button-controlled mode switching
* PWM brightness control using ESP32
* Serial Monitor status display

## Components Used

* ESP32
* LED
* Push Button

## Connections

* LED → GPIO21
* Button → GPIO33

## Output

The Serial Monitor displays the currently selected mode:

* Mode 1 : Slow Breathing
* Mode 2 : Fast Pulse
* Mode 3 : SOS Pattern

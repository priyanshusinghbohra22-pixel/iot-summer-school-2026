# Module 1: GitHub Setup & Version Control

## Overview
This module introduces the fundamentals of IoT, embedded systems, and ESP32 programming.

## Objectives
- Understand IoT concepts
- Learn ESP32 architecture
- Configure Arduino IDE
- Execute basic GPIO programs

## Experiments Included
- LED Blink
- Potentiometer Controlled LED
- Push Button Interface
- RGB LED Control

## Hardware Used
- ESP32 DevKit
- LEDs
- Resistors
- Potentiometer
- Push Button
- Breadboard
- Jumper Wires

## Software Used
- Arduino IDE
- Wokwi Simulator
- GitHub

## Learning Outcomes
After completing this module, students will be able to:
- Configure ESP32 development environment
- Interface basic peripherals
- Read analog inputs
- Control digital outputs
- Use serial monitoring

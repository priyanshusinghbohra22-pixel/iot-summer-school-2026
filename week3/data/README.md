# Environmental Monitoring Station

This project monitors environmental conditions using a **DHT22 sensor** and displays readings in CSV format.

### Output Parameters

* Timestamp (ms)
* Temperature in Celsius (°C)
* Temperature in Fahrenheit (°F)
* Humidity (%)

---

# Multi-Sensor Data Logger

This project integrates multiple sensors to collect and log environmental data.

### Output Parameters

* Temperature (°C)
* Humidity (%)
* Light Intensity (%)
* Distance (cm)

Sensor readings are displayed on the Serial Monitor every 5 seconds.

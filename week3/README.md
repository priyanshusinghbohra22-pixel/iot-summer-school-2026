# Module 3 – Sensor Interfacing using ESP32

This module focused on learning how to interface different sensors and actuators with the ESP32 and process real-time data for IoT applications.

## Topics Covered

* Temperature and Humidity Sensing using DHT11 and DHT22
* Distance Measurement using HC-SR04 Ultrasonic Sensor
* Ambient Light Detection using LDR
* Motion Detection using PIR Sensor
* Servo Motor Control using PWM
* Analog Sensor Reading and Calibration
* LED Control and Status Indication
* Buzzer-Based Alert Systems
* Data Logging through Serial Monitor
* CSV Data Formatting
* PWM Brightness Control
* Event-Based Automation
* Non-Blocking Programming using `millis()`
* Multi-Sensor Integration
* Environmental Monitoring Applications
* Smart Lighting and Parking Assistance Systems

## Learning Outcomes

By completing this module, we gained practical experience in:

* Reading digital and analog sensor data
* Interfacing multiple sensors with ESP32
* Implementing automation logic
* Monitoring environmental parameters
* Controlling actuators such as LEDs, buzzers, and servo motors
* Designing basic IoT-based monitoring systems
